## 💼 What I've Built

### 🏭 Trace-App — Industrial Tyre Traceability System
Industrial QR-based traceability system processing **40,000+ scans daily** with 100% accurate compound mapping. Built offline-first GIS storage with MMKV and E6-encoded geocoordinates, reducing polygon storage by **~60%**. Includes RBAC, analytics dashboard, and infrared scanner integration.

### 🪙 Gold Tokenization Platform
Web3 platform for tokenizing physical gold for the MENA region. Built complete auth system with **Google Authenticator 2FA, WebAuthn passkeys, and Redis session management**. Developed a gasless crypto wallet using EIP-712 signed messages via Ethers.js. Managed shared UI and Web3 logic in a Turborepo monorepo.

### 🤖 Viginyx
Automated ADR (Adverse Drug Reaction) reporting system using **LangChain and LangGraph** with multi-step orchestration and Human-in-the-Loop validation. Integrated SSE for real-time updates and LangSmith for LLM observability.
