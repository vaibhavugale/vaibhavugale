## 🚀 Featured Work

### 🏭 Trace-App — Industrial Tyre Traceability
> React Native · Node.js · QR Scanning · APK Auto-update

Built an internal system for tracking tyre components across the production floor — **40,000+ QR scans/day** with infrared scanner integration, QR generation for industrial printing, and a closed-premise Android deployment with auto-update.

---

### 🥇 Gold Tokenization Platform
> React.js · Web3 · WebAuthn · Turborepo · KYC Integration

End-to-end onboarding with KYC for the MENA region. Multi-factor auth via **Google Authenticator 2FA + WebAuthn passkeys** with stateless token rotation. Built a **custom Web3 wallet** for gasless transactions, with shared UI managed in a Turborepo monorepo.

---

### 🗺️ EUDR Land-Mapping & Deforestation Monitor
> React Native · GeoJSON · KML · Polygon Algorithms

Complete mobile app for farm and plot mapping, built to comply with **EU Deforestation Regulation (EUDR)**. Features polygon area auto-calculation in m² and a custom **point-in-polygon algorithm** to detect plot overlaps. Exports GeoJSON and KML.

---

### ⚙️ Dynamic JSON-Driven Template Renderer
> React.js · Performance Optimisation · Caching

A React system that renders deeply nested, complex UI from raw JSON config. Optimised for minimal re-renders with selective caching — enabling non-engineers to define UI through data alone.
